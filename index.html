    About the System

    The Student Performance Prediction System uses advanced machine learning algorithms to predict academic performance based on multiple study-related factors.

    Key Parameters Analyzed

    • Study Hours: Daily time dedicated to studying (25% weight)
    • Attendance Rate: Class attendance percentage (20% weight)
    • Previous Grade: Historical academic performance (30% weight)
    • Assignment Completion: Homework completion rate (15% weight)
    • Sleep Quality: Hours of sleep per night (7% weight)
    • Extracurricular Activities: Weekly hours in activities (3% weight)

    How It Works

    The prediction model uses a trained Random Forest Classifier that has been optimized with historical student data. Each input is normalized and processed through the model to generate accurate predictions.
